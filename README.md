# ItemReward

## Adding ItemReward as a dependency to your project

### Maven

```xml

<project>
    ...
    <repositories>
        <repository>
            <id>jitpack</id>
            <url>https://jitpack.io</url>
        </repository>
        ...
    </repositories>
    ...
    <dependencies>
        <dependency>
            <groupId>com.github.la-taverne-mc</groupId>
            <artifactId>ItemReward</artifactId>
            <version>TAG</version>
            <scope>provided</scope>
        </dependency>
        ...
    </dependencies>
    ...
</project>
```
